ContentShare add buttons to share on Twitter and Facebook (Share and Like) on home page of your blog and on each post.

== Description ==

ContentShare use the API features of Twitter and Facebook to possibility add this buttons on your blog.
For more informations about this features see the API URLs of this Social Networks.

== Installation ==

* Unzip this package and upload this on your blog plugin directories.
* Configure the Content Share options on the left side menu on the admin of your blog.
* Save the configuration and...

1, 2, 3: You're done!

== Frequently Asked Questions == 
* Its possible configure the plugin to show a specific network on my blog?
Yes, you can. Its possible to manage this configurations on CMS Admin. You can change the position of share panel and enable or disable the networks to share on your blog.
